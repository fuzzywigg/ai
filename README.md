# Fuzzywigg AI

Self‑evolving AI and layered blockchain architecture for life management. Auto‑branching skills detect patterns, mint soulbound tokens, and enforce consent via L1–L4 governance. Unified dashboard integrates IoT/6G heartbeats, privacy‑first data vaults, and real‑world task triggers.

## Quickstart

### Prerequisites

- Python 3.10+
- Node.js 16+ (for contracts)

### Installation

1. **Clone the repository:**

    ```bash
    git clone https://github.com/fuzzywigg/fuzzywigg-ai.git
    cd fuzzywigg-ai
    ```

2. **Install the package:**

    ```bash
    pip install -e .
    ```

    Or just install dependencies:

    ```bash
    pip install -r requirements.txt
    ```

3. **Set up environment variables:**

    ```bash
    cp .env.example .env
    # Edit .env with your API keys
    ```

4. **Run the demo:**

    ```bash
    python demo.py
    ```
